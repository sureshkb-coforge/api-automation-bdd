# API Automation Framework

This sample framework is built using Java, RestAssured, TestNG, Cucumber, Maven, Apache POI, Log4j2, and Extent Reports.

## Included
- Working GET and PUT API scenarios against `https://reqres.in`
- Excel-driven dynamic payload generation
- External JSON payload template
- TestNG runner and Cucumber runner
- Log4j2 logging
- Extent report support

## Run commands
```bash
mvn clean test
```

## Reports
- `reports/extent-report.html`
- `reports/cucumber-report.html`
- `reports/manual-extent-report.html`
